职位诱惑：

发展前景 五险一金 萌妹爱心餐

职位描述：

岗位职责：

1）负责运营管理及效果评估工作；

2）健全运营的体系，带领团队制定和实施内容及用户运营计划；

3）把握用户需求，分析数据，通过多渠道服务用户，改善用户体验；

4）负责团队的组织管理和考核，负责部门制度、规范、流程的制定实施；

5）策划并组织活动，提高用户黏性。

 

岗位要求：

1）5年以上互联网从业经验，熟悉移动互联网娱乐及社交产品营运模式、推广渠道及媒体对接；

2）具有较全面的的策划及用户运营能力，良好的互联网娱乐感；

3）较强的项目管理能力，能够独立操作谈判项目，有谈判成功，并全程跟踪实施经验；

4）沟通能力强，掌握目标导向的工作方法，能够通过有效的手段来指导、督促团队达成目标。

运营大型社交/社区产品经验优先

 

公司简介：

一句话介绍：新生代娱乐社交入口，探索快乐的小宇宙

关键词：极致产品 新生代用户 免费午餐晚餐 节日福利 喵星人 高成长性

 

{公司产品}

「节操精选」是国内调性最好的新生代娱乐社交app，而新产品「开拍」则是以对口型为切入点的短视频娱乐社区，两者都致力于通过创新的互动体验和内容帮用户发现更快乐的世界，影响新一代的90后、00后用户，改变全民娱乐和社交的方式。

 

作为新生代娱乐社交入口，节操精选首创图文弹幕功能并获得专利，产品曾获苹果App Store官方推荐、腾讯应用宝小红花奖，并入选2014百度手机助手年度最佳应用和申万90后研究报告。产品获得广大新生代用户的喜爱，超过千万用户，目前80%用户为90后，其中30%为00后。 

{融资状况}

B轮融资完成，融资额上亿。投资人及基金包括徐小平、王啸、策源资本等。

 

{团队组合}

大牛 + 萌妹子 + 极客 + 猫咪。

 

{创始人}

陈桦，#敏感词#，江湖人称“节操姐”，毕业于中国传媒大学，连续创业者，深度热爱移动互联网产品。入选《福布斯》中国2015年30位30岁以下创业者，《创业邦》2015年值得关注的#敏感词#创业者。曾受百度、腾讯、阿里等公司邀请，分享设计新生代社交产品经验。

 

{企业文化}

geek文化，崇尚简洁设计和高效率协作，目标导向。没有表演式加班，坚持无论如何也要完成计划的任务。

给创意工作者最大的尊重和空间，不随便改改改，氛围不能再欢快更多。内部创新小组制，任何人有好的idea都可以驱动一个项目。鼓励创新。

拥抱开源技术和新技术，努力回馈开源社区，是hashids算法组织的核心成员，目前使用着Java、AWS、Docker、Nodejs和Go等平台和技术。

 

{福利}

期权；五险一金；专属爱心健康的午餐、晚餐；零食水果；定期创业营/hackthon培训和奖励；过各种节都有红包抢（除了清明节）/ 礼物；有萌妹子也有暖#敏感词#；性取向多元化；喵星人鼓励师。
